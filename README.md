# dzen
# for backend: 
### 1) cd server 
### 2) npm install 
### 3) create .env file
### example:
### port = '5000'
### host = 'localhost'
### pass_mysql = ''
### user_mysql = 'root'
### 4) run sql code: node sqlRun.mjs
### 5) npm run dev
# for frontend: 
### 1) cd client
### 2) npm install
### 3) add env variable and run project = example: 
### set "process.env.REACT_APP_LINK=http://localhost:5000/" && npm start
